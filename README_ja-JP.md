# NekokoLPA2 - 高度な eSIM マネージメント

**Language:** [English](./README.md) | **日本語**

NekokoLPA2 は、複数のカードリーダーと高度なプロファイル管理機能をサポートする強力なクロスプラットフォームな eSIM 管理アプリです。

## 機能

- **複数のリーダーに対応**: BLE、USB CCID、OMAPI、Telephony API、リモートリーダーに対応
- **カスタムプロファイルアイコン**: ギャラリー、リモートソース、eSIM カードからカスタムアイコンを設定可能
- **スマートタグシステム**: 日付のタグ、メモ、カスタムラベルを使用してプロファイルを整理できます
- **スケジュール通知**: プロファイルの有効期限や重要な日付のリマインダーを受け取れます
- **クロスプラットフォーム**: Android、iOS、macOS、Linux、Windows と Web に対応
- **高度な UI**: スマートな折りたたみアプリバー、コンパクトなコンテキストメニュー、レスポンシブデザインを採用しています
